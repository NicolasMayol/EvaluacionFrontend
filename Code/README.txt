Hola mí nombre es Nicolás Mayol, soy el desarrolador de esta pequeña app web.
El framework que elegí es NEXT.js.

Podemos comenzar clonando este repositorio o bien descargandolo como un archivo ZIP

Una vez tengamos nuestra carpeta debemos posicionarnos en la misma con nuestra terminal e introducir el comando "npm i", 
para que se descarguen e instalen todos los modulos necesarios para el correcto funcionamiento de
nuestra aplicación.

Ya instalados nuestros modulos debemos utilizar el comando "npm run dev", el cual dara inicio a nuestra pagina en la 
direccion "http://localhost:3000/".

Una vez en nuestra pagina podemos ver que tenemos la opcion de acceder a cada una de las actividades asignadas en el pdf.
Tambien se puede acceder por medio del url simplemente introduciendo los endpoints " /R2 ", " /R3 " ó " /R4 ".
